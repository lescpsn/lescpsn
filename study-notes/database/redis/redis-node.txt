[ Linux下源代码安装redis ]
  1、下载源代码
     wget http://download.redis.io/redis-stable.tar.gz

  2、解压编译安装
     > tar -zxvf redis-stable.tar.gz
     > cd redis-stable
     > make
     > make test
     > make install

  3、修改linux系统运行参数
     [ somaxconn ] : somaxconn是Linux中的一个内核参数，表示socket监听(listen)的
                     backlog(未处理请求)上限
     echo "1024" >/proc/sys/net/core/somaxconn           立马生效
     echo "net.core.somaxconn = 1024" >>/etc/sysctl.conf 重启后任然有效

     [ overcommit_memory ] : 内存分配策略
     0 : 表示内核将检查是否有足够的可用内存供应用进程使用,如果有足够的可用内存，内存申请允许,否则内存申请失败，并把错误返回给应用进程。
     1 : 表示内核允许分配所有的物理内存，而不管当前的内存状态如何。
     2 : 表示内核允许分配超过所有物理内存和交换空间总和的内存
     echo "1" >/proc/sys/vm/overcommit_memory            立马生效
     echo "vm.overcommit_memory = 1"  >>/etc/sysctl.conf   重启后任然有效

     [ transparent_hugepage ] : 关闭透明大页
     echo never  >/sys/kernel/mm/transparent_hugepage/enabled  立马生效
     重启后任然有效(ubunt和redhat该文件可能不一致)
     # vim /etc/rc.local
     # In order to enable or disable this script just change the execution
     # /etc/rc.local中写的很清楚，编辑完后需要增添文件的执行权限
       echo never >/sys/kernel/mm/transparent_hugepage/enabled
     # chmod +x /etc/rc.local
