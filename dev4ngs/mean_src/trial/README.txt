  [ApiTestArray]:
    接口开关变量,值为true表示测试该接口，其它接口不调试，可以为多个每行
    写一个
    目前只支持为true的值，方便调试
    例如:
    10000 = true
    10001 = true

  其中：10000，10001为测试接口的id（人为编写）
