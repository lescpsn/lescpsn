* Golang开发环境搭建
** Linux
  + 安装Golang
#+BEGIN_EXAMPLE
sudo apt-get install golang

bash shell:
vim ~/.bashrc
export GOPATH=~/Projects/lescpsn/golang

fish  shell:
vim  ~/.config/fish/config.fish (若没有，直接创建该文件)
set -x GOPATH ~/Projects/lescpsn/golang

mkdir $GOPATH/src
mkdir $GOPATH/bin
mkdir $GOPATH/pkg
